Введение
	В современном мире цифровые технологии играют ключевую роль в различных сферах жизни, включая финансовую. Разработка приложений для расчета ипотечных вкладов и кредитов становится всё более актуальной в условиях постоянно меняющегося рынка недвижимости и роста потребительских запросов.
Актуальность проекта заключается в создании инструмента, который поможет пользователям эффективно планировать свои финансовые обязательства, выбирать наиболее выгодные условия ипотечного кредитования и расчёта вкладов. Такое приложение будет полезным как для физических лиц, так и для банковских организаций, предлагающих услуги кредитования и депозитов.
Цель проекта – разработка приложения для расчета ипотечных вкладов и кредитов, которое позволит пользователям легко и быстро производить расчеты по ипотеке, процентным ставкам, срокам выплат и условиям вкладов. Приложение должно обеспечивать удобный интерфейс, точные расчеты и возможность сравнения различных предложений на рынке.
Для достижения этой цели необходимо выполнить следующие задачи:
– проанализировать существующие методы расчета ипотечных вкладов и кредитов;
– разработать структуру программного продукта с учетом функциональных требований;
– спроектировать и реализовать удобный пользовательский интерфейс;
– провести тестирование и валидацию корректности расчетов;
– подготовить руководство пользователя.
	Целевая аудитория проекта – физические лица, планирующие оформить ипотеку или вложить средства, а также финансовые консультанты и специалисты банковских организаций.





1 Общий раздел
1.1 Постановка задачи
	Курсовой проект ориентирован на разработку приложения для расчета ипотечных вкладов и кредитов на языке программирования C# с использованием технологии Windows Forms. Приложение предназначено для расчета ежемесячных платежей, общей суммы выплат и общей суммы процентов по ипотечным программам и кредитам. Основной целью программы является обеспечение удобного интерфейса для пользователей, который позволит им быстро и точно производить финансовые расчеты в зависимости от введенных данных.
	Программа предоставляет возможность расчета как ипотечных, так и кредитных платежей. Пользователь может ввести сумму кредита или ипотеки, процентную ставку и срок кредитования в месяцах, после чего программа автоматически выполнит расчеты и предоставит результаты. Для ипотеки предусмотрена функция выбора различных программ, таких как стандартная ипотека, военная ипотека, семейная ипотека и IT-ипотека, каждая из которых имеет свои льготные процентные ставки. Программа также выводит ежемесячный платеж, общую сумму выплат и переплату по процентам для каждой программы.
Для расчетов кредита программа выполняет аналогичные действия: пользователь вводит сумму кредита, процентную ставку и срок, а программа вычисляет ежемесячный платеж, общую сумму выплат и общую сумму переплаты по процентам. Результаты отображаются на отдельной вкладке, что позволяет удобно различать расчеты по ипотеке и кредитам.
Интерфейс программы интуитивно понятен: ввод данных осуществляется через текстовые поля, а для выполнения расчета предусмотрены соответствующие кнопки. Результаты расчета моментально отображаются на экране, что позволяет пользователю быстро оценить финансовую нагрузку в зависимости от различных параметров кредита или ипотеки.
	Интерфейс приложения разрабатывается с учетом простоты и удобства использования, с акцентом на пользователей, не имеющих глубоких знаний в области финансов. Для этого используются интуитивно понятные элементы управления, такие как вкладки для разделения функций расчета ипотеки и кредита, поля для ввода значений и кнопки для выполнения расчетов. Программа предоставляет точные результаты расчета, что помогает пользователям принять обоснованные решения при выборе кредитных продуктов.
	Разработка приложения проводится на языке программирования C# в среде Visual Studio, с использованием Windows Forms для создания графического интерфейса.

	2. Специальный раздел
	2.1 Анализ предметной области и определение функциональных требований
	Целью данного проекта является разработка приложения для расчета параметров ипотечных и кредитных продуктов. Приложение предназначено для облегчения расчетов по кредитам и ипотекам для пользователей, а также для хранения и просмотра истории вычислений.
Темой проекта является разработка приложения для расчета ипотечных вкладов и кредитов, предназначенного для:
	– Предоставление удобного интерфейса для ввода данных по ипотечным и кредитным продуктам;
	– Автоматический расчет ежемесячного платежа, общей суммы выплат и переплаты по кредитам и ипотекам;
	– Хранение истории расчетов и предоставление возможности пользователям просматривать и анализировать свои предыдущие расчеты.
	На основе анализа предметной области были определены следующие функциональные требования к приложению:
	– Приложение должно предоставлять возможность ввода данных для расчета ипотеки и кредита (сумма кредита, процентная ставка, срок кредита, тип кредита и т.д.);
	– Приложение должно автоматически рассчитывать ежемесячный платеж, общую сумму выплат и переплату по кредиту/ипотеке;
	– Приложение должно позволять пользователю выбирать тип ипотеки, что будет влиять на процентную ставку;
	– Приложение должно хранить историю всех расчетов пользователя, включая дату, параметры кредита и результаты расчетов;
	– Приложение должно позволять пользователю удалять или редактировать записи из истории расчетов;
	– Приложение должно обладать простым и интуитивным интерфейсом, доступным даже неподготовленному пользователю.
Данное приложение должно соответствовать следующим нефункциональным требованиям:

	– Приложение должно поддерживать кросс-платформенность, работая на различных операционных системах.
